# UCAS Computer Network Lab

这是中国科学院大学 (UCAS) 本科计算机系 计算机网络 (研讨课) 课程的实验代码仓库.

> [!IMPORTANT]
> 1. 本仓库中代码部分由我本人实现, 部分由 AI 完成.
> 2. 仓库中代码只保证通过对应实验的测试用例, 不保证代码正确性和鲁棒性.
> 3. 本仓库仅做学习交流用途, 思路仅供参考, **请同学们不要抄袭, 诚信作答.** 由直接照搬导致的任何可能的成绩损失后果自负.

## 目录结构

| 目录/文件 | 说明 |
| --- | --- |
| `02-fct` | 实验2: 流完成时间 (Flow Completion Time) 实验 |
| `03-socket` | 实验3: Socket 应用编程, 实现一个简单的 HTTP(S) 服务器 |
| `04-hub+switch` | 实验4: 交换机转发实验, 实现简单的集线器 (Hub) 和交换机 (Switch) |
| `05-stp` | 实验5: 生成树协议 (Spanning Tree Protocol, STP) 实验 |
| `06-bufferbloat` | 实验6: 数据包队列管理实验 |
| `07-router` | 实验7: 路由器转发实验 |
| `09-mospf` | 实验9: 链路状态路由协议 (MOSPF) 实验, 实现简单的动态路由 |
| `10-ip_lookup`| 实验10: IP 查找实验, 实现高效的 IP 查找算法 |
